4. Available Packages¶

4.1. Atomic Data Providers¶

Package Description
cherab-openadas Provides utility functions for downloading OPEN-ADAS data and loading it into CHERAB’s atomic data objects. This package is the default atomic data provider for CHERAB.
cherab-adas Provides python wrappers to the ADAS subscription packages and enables their use in CHERAB. Note that you will need an ADAS subscription to use this package. This package is currently in the early development phase and not ready for general use.

4.2. Fusion Experiment Packages¶

Package Description
cherab-aug The CHERAB configuration package for AUG.
cherab-compass The CHERAB configuration package for COMPASS.
cherab-iter Integrates CHERAB with IMAS and provides diagnostic configuration for ITER. This package is under development but not yet publicly available.
cherab-jet Experiment configuration package for JET.
cherab-mastu Experiment configuration package for MASTU.

4.3. Code Interface Packages¶

These are packages that provide interfaces to other modelling codes and workflow management tools.

Package Description
cherab-omfit Module for providing OMFIT integration and example workflow scripts.
cherab-solps Allows loading of CHERAB plasma objects from saved SOLPS simulations.
