5. Available Packages¶

5.1. Atomic Data Providers¶

Package

Description

cherab-openadas

Provides utility functions for downloading OPEN-ADAS data and loading it into Cherab’s atomic data objects. This package is the default atomic data provider for Cherab.

cherab-adas

Provides python wrappers to the ADAS subscription packages and enables their use in Cherab. Note that you will need an ADAS subscription to use this package. This package is currently in the early development phase and not ready for general use.

5.2. Fusion Experiment Packages¶

Package

Description

cherab-aug

The Cherab configuration package for AUG.

cherab-compass

The Cherab configuration package for COMPASS.

cherab-iter

Integrates Cherab with IMAS and provides diagnostic configuration for ITER. This package is under development but not yet publicly available.

cherab-jet

Experiment configuration package for JET.

cherab-mastu

Experiment configuration package for MASTU.

5.3. Code Interface Packages¶

These are packages that provide interfaces to other modelling codes and workflow management tools.

Package

Description

cherab-omfit

Module for providing OMFIT integration and example workflow scripts.

cherab-solps

Allows loading of Cherab plasma objects from saved SOLPS simulations.
