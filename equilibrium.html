5. Plasma equilibrium¶

5.1. EFIT¶

The workhorse equilibrium class.

class cherab.tools.equilibrium.efit.EFITEquilibrium¶

An object representing an EFIT equilibrium time-slice.

The EFIT data is interpolated to produced continuous functions of the equilibrium attributes, such as the magnetic flux (psi) and magnetic field.

Parameters:
  • r – EFIT grid radius axis values (array).
  • z – EFIT grid height axis values (array).
  • psi_grid – EFIT psi grid values (array).
  • psi_axis (float) – The psi value at the magnetic axis.
  • psi_lcfs (float) – The psi value at the LCFS.
  • magnetic_axis (Point2D) – The coordinates of the magnetic axis.
  • f_profile_psin – The normalised psi axis values for the current flux profile (array).
  • f_profile_magnitude – The magnitudes of the current flux profile (array).
  • b_vacuum_radius (float) – Vacuum B-field reference radius (in meters).
  • b_vacuum_magnitude (float) – Vacuum B-Field magnitude at the reference radius.
  • lcfs_polygon – An Nx2 array of (x, y) vertices specifying the LCFS boundary.
  • time (float) – The time stamp of the time-slice (in seconds).
class cherab.tools.equilibrium.efit.EFITLCFSMask¶

A 2D function that identifies if a point lies inside or outside the plasma LCFS.

This mask function returns a value of 1 if the requested point lies inside the Last Closed Flux Surface (LCFS). A value of 0.0 is returns outside the LCFS.

Parameters:
  • lcfs_polygon – An Nx2 array of (x, y) vertices specifying the LCFS boundary.
  • psi_normalised – A 2D function of normalised poloidal flux.
class cherab.tools.equilibrium.efit.EFITMagneticField¶

A 2D magnetic field vector function derived from EFIT data.

Parameters:
  • psi_normalised – A 2D function of normalised poloidal flux.
  • dpsi_dr – A 2D function of the radius differential of poloidal flux.
  • dpsi_dz – A 2D function of the height differential of poloidal flux.
  • f_profile – A 1D function containing a current flux profile.
  • b_vacuum_radius – Vacuum B-field reference radius (in meters).
  • b_vacuum_magnitude – Vacuum B-Field magnitude at the reference radius.
  • inside_lcfs – A 2D mask function returning 1 if inside the LCFS and 0 otherwise.
